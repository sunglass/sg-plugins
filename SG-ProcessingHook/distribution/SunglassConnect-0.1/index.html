SunglassConnect

A library by Kaustuv DeBiswas for the Processing programming environment.
Last update, 07/13/2012.

A collection of utilities for connecting with Sunglass, a cloud based 3d platform.
Download / Upload / Manage your 3D data via HTTPS using the Sunglass REST API.

Download

Download SunglassConnect version 0.1.1 (0.1) in .zip format.

Installation

Unzip and put the extracted SunglassConnect folder into the libraries folder of your Processing sketches. Reference and examples are included in the SunglassConnect folder.
